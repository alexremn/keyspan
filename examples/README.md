# keyspan examples

A self-contained demo: a gitleaks finding becomes the entry point into a
cross-surface blast-radius graph. Everything here is reproducible.

## Layout

- `repo/` — a tiny demo repository (GitHub Actions workflow, CODEOWNERS, k8s +
  ExternalSecret manifest).
- `reports/gitleaks.json` — a sample gitleaks report (fake values only).
- `outputs/` — committed sample outputs (`blast-radius.{txt,json,html}`),
  regenerated by `scripts/reproduce_examples.sh`.
- `demo.cast` — an asciinema v2 cast of the quickstart.

## 60-second quickstart

```bash
go build -o keyspan ./cmd/keyspan

./keyspan ingest gitleaks examples/reports/gitleaks.json
./keyspan scan examples/repo
./keyspan recorrelate
./keyspan blast-radius name:DATABASE_PASSWORD
```

You should see `DATABASE_PASSWORD` linked to a GitHub Actions step, a Kubernetes
Deployment, and an ExternalSecret, each with a confidence band and owners.

## Re-record the cast (optional)

```bash
asciinema rec examples/demo.cast \
  --title "keyspan demo: gitleaks finding -> blast radius" \
  --command "bash -lc 'keyspan ingest gitleaks examples/reports/gitleaks.json; keyspan scan examples/repo; keyspan recorrelate; keyspan blast-radius name:DATABASE_PASSWORD'"
```

## Verify reproduction

```bash
./scripts/reproduce_examples.sh
```
